# Autumn

面向UNITY的轻量级、渐进式、开箱即用的IOC框架。

- 轻量级：AutumnCore核心代码仅占约400行。
- 渐进式：在你需要的任何地方、任何时机引入并享用。
- 开箱即用：引入Autumn后，零配置，零代码，通过[特性]与框架交互。
- 可扩展：你可以自己编写插件，来改变Autumn组织Bean的方式。
